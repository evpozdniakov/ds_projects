**–î–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ—Å—Ç—å** ‚Äî¬†—ç—Ç–æ —Å–∏—Ç—É–∞—Ü–∏—è –∫–æ–≥–¥–∞ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –¥–µ–π—Å—Ç–≤–∏–µ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ –æ–¥–Ω–æ–º—É –∏ —Ç–æ–º—É –∂–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É.

**–°–ª—É—á–∞–π–Ω—ã —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç** ‚Äî¬†—ç—Ç–æ –¥–µ–π—Å—Ç–≤–∏–µ, —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–µ –¥–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω.

**–≠–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã–π –∏—Å—Ö–æ–¥** ‚Äî —ç—Ç–æ –æ–¥–∏–Ω (–ª—é–±–æ–π) –∏–∑ –≤—Å–µ—Ö –≤–æ–∑–º–æ–∂–Ω—ã—Ö —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–≤ —Å–ª—É—á–∞–π–Ω–æ–≥–æ —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–∞.

**–ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –∏—Å—Ö–æ–¥–æ–≤** ‚Äî¬†—ç—Ç–æ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –∏—Å—Ö–æ–¥–æ–≤ –∫–∞–∫–æ–≥–æ-—Ç–æ –æ–¥–Ω–æ–≥–æ —Å–ª—É—á–∞–π–Ω–æ–≥–æ —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–∞. –û–Ω–æ –æ–±—ã—á–Ω–æ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è –±—É–∫–≤–æ–π `Œ©` (–æ–º–µ–≥–∞).

**–î–æ—Å—Ç–æ–≤–µ—Ä–Ω–æ–µ —Å–æ–±—ã—Ç–∏–µ** ‚Äî —ç—Ç–æ —Ç–æ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–æ—á–Ω–æ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç (–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å `1`)

**–ù–µ–≤–æ–∑–º–æ–∂–Ω–æ–µ —Å–æ–±—ã—Ç–∏–µ** ‚Äî —ç—Ç–æ —Ç–æ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–æ—á–Ω–æ –Ω–µ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç (–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å `0`)

**–î–æ–ø–æ–ª–Ω–µ–Ω–∏–µ–º** —Å–æ–±—ã—Ç–∏—è `A` –Ω–∞–∑—ã–≤–∞—é—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏—Å—Ö–æ–¥–æ–≤, –Ω–µ –±–ª–∞–≥–æ–ø—Ä–∏—è—Ç—Å—Ç–≤—É—é—â–∏—Ö —Å–æ–±—ã—Ç–∏—é `A`. –û–±–æ–∑–Ω–∞—á–∞—é—Ç `ƒÄ`.

**–ù–µ—Å–æ–≤–º–µ—Å—Ç–∏–º—ã–µ —Å–æ–±—ã—Ç–∏—è** ‚Äî —ç—Ç–æ —Å–æ–±—ã—Ç–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –º–æ–≥—É—Ç –ø–µ—Ä–µ—Å–µ—á—å—Å—è. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–æ–∂–¥—å –ø–æ–π–¥–µ—Ç –∏ –¥–æ–∂–¥—å –Ω–µ –ø–æ–π–¥–µ—Ç.

**–ê–ø—Ä–∏–æ—Ä–Ω—ã–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏** ‚Äî —ç—Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —Ç–∞–∫–∏—Ö —Å–æ–±—ã—Ç–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –∑–∞–≤–∏—Å—è—Ç –æ—Ç –Ω–∞—Å—Ç—É–ø–ª–µ–Ω–∏—è –¥—Ä—É–≥–∏—Ö, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è —ç—Ç–æ *–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –±–µ–∑—É—Å–ª–æ–≤–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π*.

**–ê–ø–æ—Å—Ç–µ—Ä–∏–æ—Ä–Ω—ã–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏** ‚Äî¬†—ç—Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —Å–æ–±—ã—Ç–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –∑–∞–≤–∏—Å—è—Ç –æ—Ç –¥—Ä—É–≥–∏—Ö, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è —ç—Ç–æ *–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —É—Å–ª–æ–≤–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π*.

`P(A) + P(ƒÄ) = 1`

### –ü—Ä–∞–≤–∏–ª–∞

#### –ü—Ä–∞–≤–∏–ª–æ —Å—É–º–º—ã

–î–ª—è –¥–≤—É—Ö –≤–∑–∞–∏–º–æ–∏—Å–∫–ª—é—á–∞—é—â–∏—Ö —Å–æ–±—ã—Ç–∏–π `A` –∏ `B` –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –Ω–∞—Å—Ç—É–ø–ª–µ–Ω–∏—è –ª—é–±–æ–≥–æ –∏–∑ –Ω–∏—Ö —Ä–∞–≤–Ω–∞ —Å—É–º–º–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π –∫–∞–∂–¥–æ–≥–æ –∏–∑ –Ω–∏—Ö.

`P(A ‚à™ B) = P(A) + P(B)`

–§–æ—Ä–º—É–ª—É —ç—Ç–æ–≥–æ –ø—Ä–∞–≤–∏–ª–∞ –ª–µ–≥–∫–æ –≤—ã–≤–µ—Å—Ç–∏:

#### –ü—Ä–∞–≤–∏–ª–æ —É–º–Ω–æ–∂–µ–Ω–∏—è

#### –û–±–æ–±—â–µ–Ω–Ω–æ–µ –ø—Ä–∞–≤–∏–ª–æ —Å—É–º–º—ã

### –£—Å–ª–æ–≤–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å
