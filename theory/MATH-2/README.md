**–î–µ—Ç–µ—Ä–º–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ—Å—Ç—å** ‚Äî¬†—ç—Ç–æ —Å–∏—Ç—É–∞—Ü–∏—è –∫–æ–≥–¥–∞ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –¥–µ–π—Å—Ç–≤–∏–µ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ –æ–¥–Ω–æ–º—É –∏ —Ç–æ–º—É –∂–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É.

**–î–æ—Å—Ç–æ–≤–µ—Ä–Ω–æ–µ —Å–æ–±—ã—Ç–∏–µ** ‚Äî —ç—Ç–æ —Ç–æ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–æ—á–Ω–æ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç (–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å `1`)

**–ù–µ–≤–æ–∑–º–æ–∂–Ω–æ–µ —Å–æ–±—ã—Ç–∏–µ** ‚Äî —ç—Ç–æ —Ç–æ, –∫–æ—Ç–æ—Ä–æ–µ —Ç–æ—á–Ω–æ –Ω–µ –ø—Ä–æ–∏–∑–æ–π–¥–µ—Ç (–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å `0`)

**–î–æ–ø–æ–ª–Ω–µ–Ω–∏–µ–º** —Å–æ–±—ã—Ç–∏—è `A` –Ω–∞–∑—ã–≤–∞—é—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏—Å—Ö–æ–¥–æ–≤, –Ω–µ –±–ª–∞–≥–æ–ø—Ä–∏—è—Ç—Å—Ç–≤—É—é—â–∏—Ö —Å–æ–±—ã—Ç–∏—é `A`. –û–±–æ–∑–Ω–∞—á–∞—é—Ç `ƒÄ`.

**–ù–µ—Å–æ–≤–º–µ—Å—Ç–∏–º—ã–µ —Å–æ–±—ã—Ç–∏—è** ‚Äî —ç—Ç–æ —Å–æ–±—ã—Ç–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –º–æ–≥—É—Ç –ø–µ—Ä–µ—Å–µ—á—å—Å—è. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–æ–∂–¥—å –ø–æ–π–¥–µ—Ç –∏ –¥–æ–∂–¥—å –Ω–µ –ø–æ–π–¥–µ—Ç.

**–ê–ø–æ—Å—Ç–µ—Ä–∏–æ—Ä–Ω—ã–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏** ‚Äî¬†—ç—Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —Å–æ–±—ã—Ç–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –∑–∞–≤–∏—Å—è—Ç –æ—Ç –¥—Ä—É–≥–∏—Ö, –∏–Ω–∞—á–µ –≥–æ–≤–æ—Ä—è —ç—Ç–æ *–≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —É—Å–ª–æ–≤–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π*.

### –ü—Ä–∞–≤–∏–ª–∞

#### –ü—Ä–∞–≤–∏–ª–æ —Å—É–º–º—ã

–î–ª—è –¥–≤—É—Ö –≤–∑–∞–∏–º–æ–∏—Å–∫–ª—é—á–∞—é—â–∏—Ö —Å–æ–±—ã—Ç–∏–π `A` –∏ `B` –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –Ω–∞—Å—Ç—É–ø–ª–µ–Ω–∏—è –ª—é–±–æ–≥–æ –∏–∑ –Ω–∏—Ö —Ä–∞–≤–Ω–∞ —Å—É–º–º–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π –∫–∞–∂–¥–æ–≥–æ –∏–∑ –Ω–∏—Ö.

–§–æ—Ä–º—É–ª—É —ç—Ç–æ–≥–æ –ø—Ä–∞–≤–∏–ª–∞ –ª–µ–≥–∫–æ –≤—ã–≤–µ—Å—Ç–∏:

#### –û–±–æ–±—â–µ–Ω–Ω–æ–µ –ø—Ä–∞–≤–∏–ª–æ —Å—É–º–º—ã

### –£—Å–ª–æ–≤–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å

–ó–¥–µ—Å—å `x·µ¢` ‚Äî —ç—Ç–æ –æ–¥–Ω–æ –∏–∑ –∑–Ω–∞—á–µ–Ω–∏–π, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–∂–µ—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å —Å–ª—É—á–∞–π–Ω–∞—è –≤–µ–ª–∏—á–∏–Ω–∞, –∞ `P(X = x·µ¢)` ‚Äî –µ–≥–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å.

#### –ü—Ä–∏–º–µ—Ä

–°–ª—É—á–∞–π–Ω–∞—è –≤–µ–ª–∏—á–∏–Ω–∞ –∑–∞–¥–∞–Ω–∞ —Ç–∞–±–ª–∏—Ü–µ–π

| X |-1 | 0 | 1 | 2 |
|---|---|---|---|---|
| P |0.5|0.2|0.2|0.1|

–ù—É–∂–Ω–æ –Ω–∞–π—Ç–∏ `E(X)` –∏ `E(X¬≤)`

```
E(X) = -1*0.5 + 0*0.2 + 1*0.2 + 2*0.1 = -0.1
E(X¬≤) = 1*0.5 + 0*0.2 + 1*0.2 + 4*0.1 = 1.1
```

‚ùóÔ∏è –ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –æ–∂–∏–¥–∞–Ω–∏–µ –æ—á–µ–Ω—å —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –∫ –≤—ã–±—Ä–æ—Å–∞–º.

### –î–∏—Å–ø–µ—Ä—Å–∏—è –∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ

–ò–º–µ—Ç—å —Ç–æ–ª—å–∫–æ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –æ–∂–∏–¥–∞–Ω–∏–µ –Ω–µ–¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ. –ù—É–∂–Ω–æ –ø–æ–Ω–∏–º–∞—Ç—å, –∫–∞–∫–æ–≤ —Ä–∞–∑–±—Ä–æ—Å —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã –≤–æ–∫—Ä—É–≥ –æ–∂–∏–¥–∞–Ω–∏—è. –î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ –∏ –¥–∏—Å–ø–µ—Ä—Å–∏—é.

**–î–∏—Å–ø–µ—Ä—Å–∏—è** –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è `V(X)` –∏–ª–∏ `D(X)` –∏–ª–∏ –µ—â–µ `œÉ¬≤x` –∏ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –ø–æ —Ñ–æ—Ä–º—É–ª–µ:

`D(X) = E(X¬≤) - E(X)¬≤`

–¢–∞–∫—É—é –≤–µ–ª–∏—á–∏–Ω—É —Ç—Ä—É–¥–Ω–æ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å, –ø–æ—Ç–æ–º—É —á—Ç–æ –æ–Ω–∞ –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç —Ä—É–±–ª–∏ –≤ –∫–≤–∞–¥—Ä—Ç–∞—Ç–Ω—ã–µ —Ä—É–±–ª–∏, –∞ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–µ –º–µ—Ç—Ä—ã –≤ –º–µ—Ç—Ä—ã —á–µ—Ç–≤–µ—Ä—Ç–æ–π —Å—Ç–µ–ø–µ–Ω–∏. –ü–æ—ç—Ç–æ–º—É –¥–ª—è –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç **—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ**, –∫–æ—Ç–æ—Ä–æ–µ —è–≤–ª—è–µ—Ç—Å—è –∫–æ—Ä–Ω–µ–º –∏–∑ –¥–∏—Å–ø–µ—Ä—Å–∏–∏ –∏ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è `œÉx`.

`œÉx = ‚àöD(X)`

‚ùóÔ∏è –î–∏—Å–ø–µ—Ä—Å–∏—è –∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ –æ—á–µ–Ω—å —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã –∫ –≤—ã–±—Ä–æ—Å–∞–º.

### –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –¥–∏—Å–∫—Ä–µ—Ç–Ω–æ–π —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã

–°–ª—É—á–∞–π–Ω—ã–µ –≤–µ–ª–∏—á–∏–Ω—ã –±—ã–≤–∞—é—Ç **–¥–∏—Å–∫—Ä–µ—Ç–Ω—ã–µ** –∏ **–Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω—ã–µ**. –ò—Ö –Ω–∞–∑–≤–∞–Ω–∏–µ –≥–æ–≤–æ—Ä–∏—Ç —Å–∞–º–æ –∑–∞ —Å–µ–±—è.

–í –º–∞—Ç–µ–º–∞—Ç–∏–∫–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –æ–≥—Ä–æ–º–Ω–æ–µ –∫–æ–ª-–≤–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–π –¥–∏—Å–∫—Ä–µ—Ç–Ω–æ–π —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã, –Ω–æ –≤ —Ç–µ–æ—Ä–∏–∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π —á–∞—â–µ –≤—Å–µ–≥–æ –≤—Å—Ç—Ä–µ—á–∞—é—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–µ —á–µ—Ç—ã—Ä–µ **—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è**:

- —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ;
- —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ë–µ—Ä–Ω—É–ª–ª–∏;
- –±–∏–Ω–æ–º–∏–∞–ª—å–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ;
- —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ü—É–∞—Å—Å–æ–Ω–∞.

#### –†–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ

> –∏–≥—Ä–∞–ª—å–Ω–∞—è –∫–æ—Å—Ç—å

–ó–Ω–∞—á–µ–Ω–∏—è ‚Äî —á–∏—Å–ª–∞ –æ—Ç `1` –¥–æ `N`. –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –æ–¥–∏–Ω–∞–∫–æ–≤—ã.

#### –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ë–µ—Ä–Ω—É–ª–ª–∏

> —Ñ–∞–ª—å—à–∏–≤–∞—è –º–æ–Ω–µ—Ç–∞

–ó–Ω–∞—á–µ–Ω–∏—è `0` –∏ `1`, –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ `p` –∏ `1 - p`

#### –ë–∏–Ω–æ–º–∏–Ω–∞–ª—å–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ

> –º–∏—à–µ–Ω–∏ –≤ –±–∏–∞—Ç–ª–æ–Ω–µ

–î–ª—è —Ç–æ–≥–æ —á—Ç–æ–±—ã –æ—Ü–µ–Ω–∏—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ —Å—Ä–µ–¥–∏ `n` –∏—Å–ø—ã—Ç–∞–Ω–∏–π –±—É–¥–µ—Ç  `k` —É—Å–ø–µ—Ö–æ–≤, –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Å–ª–µ–¥—É—é—â—É—é —Ñ–æ—Ä–º—É–ª—É:

–ó–¥–µ—Å—å `(n k)` ‚Äî —ç—Ç–æ –±–∏–Ω–æ–º–∏–Ω–∞–ª—å–Ω—ã–π –∫–æ—ç—Ñ–∏—Ü–∏–µ–Ω—Ç, –∫–æ—Ç–æ—Ä—ã–π —Ä–∞–≤–µ–Ω

##### –ü—Ä–∏–º–µ—Ä

–ü—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å –≥–∞–¥–∂–µ—Ç–æ–≤ –∑–Ω–∞–µ—Ç, —á—Ç–æ `20 %` –ø—Ä–æ–∏–∑–≤–æ–¥–∏–º—ã—Ö –∏–º —Ç–æ–≤–∞—Ä–æ–≤ ‚Äî –±—Ä–∞–∫–æ–≤–∞–Ω–Ω—ã–µ. –ï—Å–ª–∏ –æ–Ω –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç –¥–µ—Å—è—Ç—å –∏–∑–¥–µ–ª–∏–π –≤ –¥–µ–Ω—å, –∫–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ –Ω–µ –±–æ–ª–µ–µ –¥–≤—É—Ö –∏–∑ –Ω–∏—Ö –±—Ä–∞–∫–æ–≤–∞–Ω–Ω—ã–µ?

```python
# P(X=8) = (10 8) * (0.8)**8 * (0.2)**2
# P(X=9) = (10 9) * (0.8)**9 * (0.2)**1
# P(X=10) = (10 10) * (0.8)**10 * (0.2)**0

# P(X={8,9,10}) = P(X=8) + P(X=9) + P(X=10)

```

#### –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ü—É–∞—Å—Å–æ–Ω–∞

–≠—Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —á–∏—Å–ª–∞ —Å–æ–±—ã—Ç–∏–π –≤ –ø–µ—Ä–∏–æ–¥ –≤—Ä–µ–º–µ–Ω–∏ —Å —É—á–µ—Ç–æ–º —Å—Ä–µ–¥–Ω–µ–≥–æ –∫–æ–ª-–≤–∞ —Å–æ–±—ã—Ç–∏–π –ø—Ä–æ–∏—Å—Ö–æ–¥—è—â–∏—Ö –∑–∞ —ç—Ç–æ—Ç –ø–µ—Ä–∏–æ–¥ –≤—Ä–µ–º–µ–Ω–∏.

##### –ü—Ä–∏–º–µ—Ä

–í —Ç–µ—á–µ–Ω–∏–µ `5` –º–∏–Ω—É—Ç –∫–æ–ª–ª-—Ü–µ–Ω—Ç—Ä –ø–æ–ª—É—á–∞–µ—Ç –≤ —Å—Ä–µ–¥–Ω–µ–º `4.5` –∑–≤–æ–Ω–∫–∞. –°–∫–æ–ª—å–∫–æ –Ω—É–∂–Ω–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤, —á—Ç–æ–±—ã –∫–ª–∏–µ–Ω—Ç—ã –≤–∏—Å–µ–ª–∏ –Ω–∞ —Ö–æ–ª–¥–µ –Ω–µ –±–æ–ª–µ–µ `10%` –≤—Ä–µ–º–µ–Ω–∏?

```python
from scipy.stats.distributions import poisson

operators_num = 1

while True:
    calls_gt_than_operators = operators_num + 1
    calls_gt_than_operators_p = poisson.pmf(calls_gt_than_operators, 4.5)

    if calls_gt_than_operators_p <= 0.1:
        break
    else:
        operators_num += 1

print(operators_num) # 6
```

### –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–π —Å–ª—É—á–∞–π–Ω–æ–π –≤–µ–ª–∏—á–∏–Ω—ã

#### –†–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ

–ü–æ–ª–µ–∑–Ω–æ –¥–ª—è –ø—Ä–æ–≥–Ω–æ–∑–∏—Ä–æ–≤–∞–Ω–∏—è —Å–ø—Ä–æ—Å–∞ –Ω–∞ —Ç–æ–≤–∞—Ä –≤ –ø–µ—Ä–≤—ã–µ –Ω–µ–¥–µ–ª–∏ –∏ –¥–ª—è –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ —Å–ª—É—á–∞–π–Ω—ã—Ö —á–∏—Å–µ–ª.

–§–æ—Ä–º—É–ª—ã —Ç–∞–∫–∏–µ:

`EX = 1 / ∆õ`

`P(X <= x) = 1 - e^-∆õx` ‚Äî –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ —Å–ª—É—á–∞–π–Ω–∞—è –≤–µ–ª–∏—á–∏–Ω–∞ –±—É–¥–µ—Ç `<= x`

–ó–¥–µ—Å—å `∆õ` ‚Äî —Å—Ä–µ–¥–Ω–µ–µ –∫–æ–ª-–≤–æ —Å–æ–±—ã—Ç–∏–π –∑–∞ –µ–¥–∏–Ω–∏—Ü—É –≤—Ä–µ–º–µ–Ω–∏

##### –ü—Ä–∏–º–µ—Ä

–í —Å—Ä–µ–¥–Ω–µ–º –Ω–æ–≤—ã–π —Ö–æ–ª–æ–¥–∏–ª—å–Ω–∏–∫ —Å–ª—É–∂–∏—Ç 10 –ª–µ—Ç. –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ –ø—Ä–æ–¥–æ–ª–∂–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤—Ä–µ–º–µ–Ω–∏ —Å–ª—É–∂–±—ã —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∞ —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ.

1. –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —Ö–æ–ª–æ–¥–∏–ª—å–Ω–∏–∫ –ø—Ä–æ—Å–ª—É–∂–∏—Ç –±–æ–ª–µ–µ 7 –ª–µ—Ç?

```python
p_lt_11 = 1 - np.exp(-1 * l * 11)
p_btw_9_and_11 = round(p_lt_11 - p_lt_9, 3)
print(f'P(–ø—Ä–æ—Å–ª—É–∂–∏—Ç > 9 –∏ < 11 –ª–µ—Ç) = {p_btw_9_and_11}') # 0.074
```

#### Bias –∏ variance

#### –ë—ç–≥–≥–∏–Ω–≥ (bagging)

–í –±–µ–≥–≥–∏–Ω–≥–µ –º—ã —Ç—Ä–µ–Ω–∏—Ä—É–µ–º –º–Ω–æ–≥–æ –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö –º–æ–¥–µ–ª–µ–π –Ω–∞ —Ä–∞–∑–Ω—ã—Ö –±—É—Ç—Å—Ç—Ä–∞–ø–∞—Ö, –∑–∞—Ç–µ–º —É—Å—Ä–µ–¥–Ω—è–µ–º –∏—Ö —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã –Ω–∞ –∫–æ–Ω–µ—á–Ω–æ–π –º–æ–¥–µ–ª–∏. –°–º—ã—Å–ª –≤ —Ç–æ–º, —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å —Ä–∞–∑–±—Ä–æ—Å–∞.

–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏ –º–æ–∂–Ω–æ –≤—ã–≤–µ—Å—Ç–∏, —á—Ç–æ –µ—Å–ª–∏ –º—ã –Ω–∞—Ç—Ä–µ–Ω–∏—Ä—É–µ–º `–ö` –º–æ–¥–µ–ª–µ–π, —Ç–æ –º—ã —É–º–µ–Ω—å—à–∏–º —Å—Ä–µ–¥–Ω–µ–∫–≤–∞–¥—Ä–∞—Ç–∏—á–µ—Å–∫—É—é –æ—à–∏–±–∫—É –≤ `–ö` —Ä–∞–∑. (–≠—Ç–æ –≤ —Ç–µ–æ—Ä–∏–∏. –ù–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ —ç—Ç–æ—Ç —ç—Ñ—Ñ–µ–∫—Ç –±—É–¥–µ—Ç –º–µ–Ω–µ–µ –≤—ã—Ä–∞–∂–µ–Ω.)


- –ë—ç–≥–≥–∏–Ω–≥ –¥–∞—ë—Ç —É–º–µ–Ω—å—à–µ–Ω–∏–µ –æ—à–∏–±–∫–∏ –≤ `–ö` —Ä–∞–∑ –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å –æ–¥–∏–Ω–æ—á–Ω–æ–π –º–æ–¥–µ–ª—å—é.
- –ë—ç–≥–≥–∏–Ω–≥ –Ω–µ —É–º–µ–Ω—å—à–∞–µ—Ç —Å–º–µ—â–µ–Ω–∏–µ –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å –æ–¥–∏–Ω–æ—á–Ω–æ–π –º–æ–¥–µ–ª—å—é.
- –ë—ç–≥–≥–∏–Ω–≥ —É–º–µ–Ω—å—à–∞–µ—Ç —Ä–∞–∑–±—Ä–æ—Å –≤ `–ö` —Ä–∞–∑ –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å –æ–¥–∏–Ω–æ—á–Ω–æ–π –º–æ–¥–µ–ª—å—é.

–í–∞–∂–Ω–æ –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ —ç—Ç–∏ —É—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏—è –±—É–¥—É—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ —Ç–æ–ª—å–∫–æ –≤ —Ç–æ–º —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –º–µ–∂–¥—É –æ—à–∏–±–∫–∞–º–∏ *–Ω—É–ª–µ–≤–∞—è –∫–æ—Ä—Ä–µ–ª—è—Ü–∏—è*.

#### Random forest

–≠—Ç–æ —Ä–∞–∑–Ω–æ–≤–∏–¥–Ω–æ—Å—Ç—å –±—ç–≥–≥–∏–Ω–≥–∞ –≤ –∫–æ—Ç–æ—Ä–æ–π –≤ –∫–∞—á–µ—Å—Ç–≤–µ –±–∞–∑–æ–≤–æ–π –º–æ–¥–µ–ª–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—é—Ç—Å—è —Ä–µ—à–∞—é—â–µ–µ –¥–µ—Ä–µ–≤–æ.

–î–ª—è —Ç—Ä–µ–Ω–∏—Ä–æ–≤–∫–∏ –¥–µ—Ä–µ–≤—å–µ–≤ –≤ —Å–ª—É—á–∞–π–Ω–æ–º –ª–µ—Å—É –≥–æ—Ç–æ–≤—è—Ç –±—É—Ç—Å—Ä—ç–ø—ã –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–≥–æ —Ä–∞–∑–º–µ—Ä–∞ –ø–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤—É –æ–±—É—á–∞—é—â–∏—Ö –ø—Ä–∏–º–µ—Ä–æ–≤ ‚Äî¬†—Ç–∞–∫ –∂–µ –∫–∞–∫ –≤ –±–µ–≥–≥–∏–Ω–≥–µ.

–ß—Ç–æ –∫–∞—Å–∞–µ—Ç—Å—è –ø—Ä–∏–∑–Ω–∞–∫–æ–≤, —Ç–æ –±–µ—Ä—É—Ç –Ω–µ –≤—Å–µ –∏–º–µ—é—â–∏–µ—Å—è `M`, –∞ —Ç–æ–ª—å–∫–æ —á–∞—Å—Ç—å `L`:

‚Äî —ç—Ç–∞ —Ç–∞–∫–∞—è –º–µ—Ç—Ä–∏–∫–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ—Ü–µ–Ω–∏—Ç—å –∫–∞—á–µ—Å—Ç–≤–æ –∏—Ç–æ–≥–æ–≤–æ–π –º–æ–¥–µ–ª–∏ –±–µ–∑ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —Ç–µ—Å—Ç–æ–≤–æ–π –≤—ã–±–æ—Ä–∫–∏.

–ß—Ç–æ–±—ã —ç—Ç–æ —Å–¥–µ–ª–∞—Ç—å –Ω—É–∂–Ω–æ –ø—Ä–æ—Ç–µ—Å—Ç–∏—Ä–æ–≤–∞—Ç—å –∫–∞–∂–¥–æ–µ –¥–µ—Ä–µ–≤–æ –Ω–∞ –ø—Ä–∏–º–µ—Ä–∞—Ö, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ –ø–æ–ø–∞–ª–∏ –≤ –æ–±—É—á–∞—é—â—É—é –≤—ã–±–æ—Ä–∫—É –º–æ–¥–µ–ª–∏. (–ü–æ —Ç–µ–æ—Ä–∏–∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –≤ –æ–±—É—á–∞—é—â—É—é –≤—ã–±–æ—Ä–∫—É –∫–∞–∂–¥–æ–≥–æ –¥–µ—Ä–µ–≤–∞ –ø–æ–ø–∞–¥–∞–µ—Ç –ª–∏—à—å `63.2%` —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö –Ω–∞–±–ª—é–¥–µ–Ω–∏–π, –∞ `36.8%` —Å–æ—Å—Ç–∞–≤–ª—è—é—Ç out-of-bag –≤—ã–±–æ—Ä–∫—É.) –≠—Ç–æ —Å–≤–æ–µ–≥–æ —Ä–æ–¥–∞ k-folding.
