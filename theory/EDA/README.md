#### –ü–æ—Ä—è–¥–∫–æ–≤–æ–µ –∫–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ (Ordinal encoding)

### –ü—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ (Feature transformation)

–í EDA –¥–∞–Ω–Ω—ã–µ —á–∞—Å—Ç–æ –ø–æ–¥–≤–µ—Ä–≥–∞—é—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è–º, —á—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å –æ–±—É—á–µ–Ω–∏–µ –º–æ–¥–µ–ª–∏ –±–æ–ª–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã–º. –°–∞–º—ã–µ —á–∞—Å—Ç—ã–µ –≤–∏–¥—ã –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π —ç—Ç–æ **–Ω–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è** –∏ **—Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∞—Ü–∏—è** (–∏—Ö —Ç–∞–∫–∂–µ –Ω–∞–∑—ã–≤–∞—é—Ç *—à–∫–∞–ª–∏—Ä–æ–≤–∞–Ω–∏–µ–º*).

#### –ù–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è

–ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º —á—Ç–æ —É –Ω–∞—Å –µ—Å—Ç—å —Ç–∞–∫–∏–µ –¥–∞–Ω–Ω—ã–µ –∏ –ø–æ—Å–º–æ—Ç—Ä–∏–º –∫–∞–∫ –æ–Ω–∏ –±—É–¥—É—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω—ã —ç—Ç–∏–º –¥–≤—É–º—è –º–µ—Ç–æ–¥–∞–º–∏.

##### MinMaxScaler
–ê–ª–≥–æ—Ä–∏—Ç–º:
1. –ò–∑ –∫–∞–∂–¥–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –ø—Ä–∏–∑–Ω–∞–∫–∞ `x` –≤—ã—á–∏—Ç–∞–µ—Ç—Å—è –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ\
`x - x_min`
2. –†–µ–∑—É–ª—å—Ç–∞—Ç –¥–µ–ª–∏—Ç—Å—è –Ω–∞ —Ä–∞–∑–Ω–∏—Ü—É –º–µ–∂–¥—É –º–∞–∫—Å–∏–º—É–º–æ–º –∏ –º–∏–Ω–∏–º—É–º–æ–º –ø—Ä–∏–∑–Ω–∞–∫–∞\
`(x - x_min) / (x_max - x_min)`

![](./images/dst-eda-3-18.png)

```python
# create scaler once and fit it to some initial data
scaler = preprocessing.MinMaxScaler().fit(data[columns_to_scale])
# use the scaler for data (once or many times)
ndarr = scaler.transform(data[columns_to_scale])
# create a DataFrame out of ndarr
norm_df = pd.DataFrame(ndarr, columns=[columns_to_scale]).set_index(data.index)
# update the transformed columns data
data.loc[:, columns_to_scale] = norm_df[columns_to_scale]
```

### –û—Ç–±–æ—Ä –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ (Feature selection)

–î–ª—è —á–µ–≥–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –æ—Ç–±–∏—Ä–∞—Ç—å –ø—Ä–∏–∑–Ω–∞–∫–∏?

- —á—Ç–æ–±—ã –ø–æ–≤—ã—Å–∏—Ç—å –∫–∞—á–µ—Å—Ç–≤–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–π

–ö–∞–∫–∏–µ –µ—Å—Ç—å —Å–ø–æ—Å–æ–±—ã –æ—Ç–±–æ—Ä–∞ –ø—Ä–∏–∑–Ω–∞–∫–æ–≤?
- —É–¥–∞–ª–µ–Ω–∏–µ —Å–∏–ª—å–Ω–æ —Å–∫–æ—Ä—Ä–µ–ª–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ (–æ–¥–Ω–æ–≥–æ –∏–∑ –∫–∞–∂–¥–æ–π –ø–∞—Ä—ã)
- –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –Ω–∞–∏–±–æ–ª–µ–µ –∑–Ω–∞—á–∏–º—ã—Ö –ø—Ä–∏–∑–Ω–∞–∫–æ–≤

- –ª–µ–≥—á–µ –±—É–¥–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤–¥–∞–ª—å–Ω–µ–π—à–µ–º (–Ω–µ –Ω—É–∂–Ω–æ –æ–∫—Ä—É–≥–ª—è—Ç—å, –Ω–æ—Ä–º–∞–ª–∏–∑–æ–≤–∞—Ç—å –∏ –ø—Ä.)

## –°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–µ —Ç–µ—Å—Ç—ã

### –ü–æ–Ω—è—Ç–∏–µ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–æ–π –≥–∏–ø–æ—Ç–µ–∑—ã

**–°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∞—è –≥–∏–ø–æ—Ç–µ–∑–∞** —ç—Ç–æ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏–µ –æ –Ω–∞–ª–∏—á–∏–∏ —Å–≤—è–∑–∏ –º–µ–∂–¥—É –¥–≤—É–º—è –ø—Ä–∏–∑–Ω–∞–∫–∞–º–∏.

#### –ù—É–ª–µ–≤–∞—è –∏ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–∞—è –≥–∏–ø–æ—Ç–µ–∑—ã
–ß—Ç–æ–±—ã –ø–æ–¥—Ç–≤–µ—Ä–¥–∏—Ç—å –≥–∏–ø–æ—Ç–µ–∑—É –Ω—É–∂–Ω–æ –µ–µ —Å—Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–∞—Ç—å –≤ –≤–∏–¥–µ –¥–≤—É—Ö –≥–∏–ø–æ—Ç–µ–∑, –ø—Ä–æ—Ç–∏–≤–æ—Ä–µ—á–∞—â–∏—Ö –¥—Ä—É–≥-–¥—Ä—É–≥—É:
- **–Ω—É–ª–µ–≤–∞—è –≥–∏–ø–æ—Ç–µ–∑–∞** (H‚ÇÄ) –¥–æ–ª–∂–Ω–∞ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞—Ç—å –æ—Ç—Å—É—Ç—Å—Ç–≤–∏–µ –≤–∑–∞–∏–º–æ—Å–≤—è–∑–∏
- **–∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–∞—è –≥–∏–ø—Ç–æ–µ–∑–∞** (H‚ÇÅ) –¥–æ–ª–∂–Ω–∞ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞—Ç—å –Ω–∞–ª–∏—á–∏–µ –≤–∑–∞–∏–º–æ—Å–≤—è–∑–∏

–ó–∞—Ç–µ–º –Ω—É–∂–Ω–æ –ø–æ–ø—ã—Ç–∞—Ç—å—Å—è –æ–ø—Ä–æ–≤–µ—Ä–≥–Ω—É—Ç—å H‚ÇÄ. –ï—Å–ª–∏ —ç—Ç–æ —É–¥–∞—Å—Ç—Å—è, –∑–Ω–∞—á–∏—Ç –Ω–∞—à–∞ –≥–∏–ø–æ—Ç–µ–∑–∞ H‚ÇÅ –≤–µ—Ä–Ω–∞.

#### –û—à–∏–±–∫–∏ I –∏ II —Ä–æ–¥–∞

–ß—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –≤–µ—Ä–Ω–∞ –∏–ª–∏ –Ω–µ—Ç H‚ÇÄ –Ω–∞–º –ø–æ—Ç—Ä–µ–±—É–µ—Ç—Å—è –ø—Ä–æ–∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å –∏–º–µ—é—â–∏–µ—Å—è –¥–∞–Ω–Ω—ã–µ. –ü–æ—Å–∫–æ–ª—å–∫—É –º—ã –±—É–¥–µ–º —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ —á–∞—Å—Ç—å –¥–∞–Ω–Ω—ã—Ö, –∞ –Ω–µ –≤—Å–µ (–ø–æ—Å–∫–æ–ª—å–∫—É —ç—Ç–æ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ), —Ç–æ –º—ã –º–æ–∂–µ–º –¥–æ–ø—É—Å—Ç–∏—Ç—å –æ–¥–Ω—É –∏–∑ –¥–≤—É—Ö –æ—à–∏–±–æ–∫:
